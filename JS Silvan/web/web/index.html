My cheatSheet

cmd

  • cd: change directory
  • cd (naam bestand)/cd (naam bestand) etc.
  • cd .. :dan ga je één stap terug
  • cd ../.. : dan ga je meerdere stappen terug
  • mkdir: Make new directory

git

  • git status: geeft de status van de repository
  • git init: begint een nieuwe git repository
  • git add "(bestandsnaam)" of git add . (dan wordt alles in 1x toegevoegd)
  • git log: dan zie je alle veranderingen
  • git log --oneline: dan wordt alles op 1 lijn gezet (is wat overzichtelijker)
  • git push: dan push je het naar Github toe
  • git pull: dan haal je alle veranderingen op
